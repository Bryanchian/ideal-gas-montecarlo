# Ideal Gas Monte Carlo Simulation

Monte Carlo simulation of a one-dimensional ideal gas using the Metropolis algorithm.

## Features

- Metropolis Monte Carlo simulation
- Energy vs Temperature analysis
- Velocity distribution evolution
- Comparison with theoretical predictions

## Requirements

- Python 3
- numpy
- matplotlib

## Running the simulation

Run the Jupyter notebook:

ideal_gas_montecarlo.ipynb
